# Api for authentication system in djangorestframework-simplejwt and drf_social_oauth2

  ##### 1-Register user
    http://127.0.0.1:8000/register/
  ##### 2-Login
    http://127.0.0.1:8000/login/
  ##### 3-Change_password
    http://127.0.0.1:8000/change_password/
  ##### 4-Reset_password
    http://127.0.0.1:8000/reset-password/<uid>/<token>/
  ##### 5-Logout
    http://127.0.0.1:8000/logout/
  ##### 6-Google register/login
    http://127.0.0.1:8000/auth/convert-token/
  ##### 7-Github register/login
    http://127.0.0.1:8000/auth/convert-token/
